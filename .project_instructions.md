# KTRACE Analyzer

## Overview
Web app for analyzing KTRACE log files from call center software. Tracks where and how calls end across script pages.

## Technical Approach
- **Parsing**: Client-side KTRACE log parsing (no backend/database needed)
- **State**: In-memory aggregation via React state, supports multiple log uploads
- **Charts**: recharts for grouped bar chart visualization
- **UI**: Tailwind CSS with shadcn/ui components, Framer Motion animations

## Log Parsing Logic
1. Scan lines for "Agent Key Press - '+'" (plus_key) and "Received HTTP Message: protocall_stop" (protocall_stop)
2. Track last call-end trigger type
3. On encountering non-empty SESSION_DATAFIELD_HOTKEYSEQUENCEBYPAGE, find page segment containing "+"
4. Associate page with the last call-end trigger type
5. Aggregate counts per page across multiple uploaded files

## Color Scheme
- Primary: Purple (hsl 262 83% 58%)
- Chart colors: #60B5FF (+ Key), #FF9149 (Protocall Stop)
- Light theme default

## Tabs
The app has two top-level tabs (`components/ktrace-analyzer.tsx` shell):
1. **Call End Analysis** (`components/call-end-analysis.tsx`) — original client-side
   call-end tracking via `lib/ktrace-parser.ts`.
2. **Timing Analysis** (`components/timing-analysis.tsx`) — integrates `analyzer.py`.
   Uploads logs to `app/api/analyze/route.ts`, which runs `python/run_analysis.py`
   (parser.py + analyzer.py + stats.py). Detects and displays each file's ktrace
   **level**, shows a prominent "Analysis is only necessary for Level 3 ktraces"
   notice, and presents the five human-derived timing attributes per Level → Page →
   Message. Requires Python 3 on PATH (override via `PYTHON_BIN`).

## Key Files
- `lib/ktrace-parser.ts` - Core parsing & aggregation logic
- `components/ktrace-analyzer.tsx` - Main orchestrator component
- `components/call-end-chart.tsx` - recharts bar chart
- `components/data-table.tsx` - Sortable results table
- `components/file-uploader.tsx` - Drag & drop file upload
- `components/stats-cards.tsx` - Summary stat cards with animated numbers
- `components/file-list.tsx` - Uploaded file list display
